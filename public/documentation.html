myFlix App

Create a REST API with Node.js

Objective

To build the server-side component of the "myFlix" web application

The web application will provide users with access to informatoin about different movies, directors, and genres.

Out users will be able to sign up, update their personal information, and create a list of favorite movies



Request Url Endpoint HTTP Method Request Body Request Example Response Body Response Body Example
Get a list of all movies /movies GET None /movies JSON A JSON object holding data about all the movies
Get information of one specific movie by title, use %20 for spaces /movies/:Title GET None /movies/Parasite JSON {
title: "Parasite",
genre: "thriller",
director: "Bong Joon-Ho"
}
Return data about genre of a movie by its title, use %20 for spaces /movies/Genres/:Name GET None /movies/Genres/Horror JSON {
name: "horror",
description: "Horror is a genre of film that seeks to elicit fear."
}
Return data about a director by name, use %20 for spaces /movies/Directors/:Name GET None /movies/Directors/Martin%20Scorsese JSON {
name: "Martin Scorsese",
careerSpan: "1967-present"
}
Allow new users to register /users POST JSON {
name: "Francisco Trincao",
favoriteGenre: "comedy"
} 		JSON {
id: 5,
name: "Francisco Trincao",
favoriteGenre: "comedy"
}
Allow users to update their registration information /users/:Username PUT JSON {
name: "Francisco Trincao",
favoriteGenre: "western"
} 		JSON {
id: 5,
name: "Francisco Trincao",
favoriteGenre: "western"
}
Allows users to add a movie to list of favorite movies /users/:Username/Movies/:MovieID POST None /users/carlosflores/Movies/1x1x1x1x1x1x Message "Movie successfully added to your favourites"
Allow users to remove a movie from their favourites list /users/:Username/Movies/:MovieID DELETE None /users/carlosflores/Movies/2y2y2y2y2y2y Message "Movie successfully removed from your favourites"
Allow existing users to de-register /users/:Username DELETE None /users/carlosflores Message "carlosflores was successfully de-registered from myFlix"
